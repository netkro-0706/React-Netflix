## 개발목표  
코딩알려주는 누나의 수업과정으로  
다양한 API와 라이브러리를 사용하여 넷플릭스와 같은 홈페이지를 구축

## 기능  
메인화면에서 인기있는 영화 리스트, 가장 평점 좋은 영화 리스트, 상영 예정작을 볼 수 있다.  
상세정보에서 영화의 제목, 평점, 줄거리 요약등 상세정보를 볼 수 있다.  
영화의 검색이 가능하다.  
영화의 정보를 정렬하거나 원하는 장르를 선택하여 필터링 할 수 있다.  
화면의 크기에 맞춰 표시되도록 반응형 웹으로 제작.
